# Psicología

Sitio web profesional de **Psic. Fernando Fierro**.

Espacio de psicoeducación clara, directa y humana sobre:
- Adicciones
- Vínculo materno y dinámicas familiares
- Reconstrucción emocional
- Conciencia, responsabilidad y proceso personal

## Enfoque
El contenido publicado tiene fines **informativos y psicoeducativos**.  
No sustituye procesos terapéuticos ni evaluaciones clínicas individuales.

## Tecnología
Proyecto web estático desarrollado con:
- HTML5
- CSS3
- JavaScript
- GitHub Pages
